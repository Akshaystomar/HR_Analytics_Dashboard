# HR_Analytics_Dashboard
power bi project

Project Objective: The objective of this project is to analyse employee attendance, working hours, performance and leave patterns to help the HR team understand workforce behaviour and improve their daily decision-making. The dashboard gives quick insights that help save time, identify issues early, and make HR processes more efficient.

What I Did:
Loaded and cleaned the dataset directly in Power BI (Power Query)
Fixed date formats and created new DAX columns like WFH count, SL count and Day of Week
Created new measures like Presence %, SL %, and Working Hours
Designed an interactive and easy-to-read dashboard for HR use
Highlighted important employee trends using charts and KPIs

Tools Used: Power BI,
Power Query,
DAX,
Excel (dataset)
